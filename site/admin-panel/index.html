    Admin Panel¶

    Master the administrative features of My Attend. This comprehensive guide covers everything administrators need to effectively manage their organization's attendance system.

    Getting Started as an Admin¶

    Admin Dashboard Overview¶

    When you log in as an administrator, you'll see a powerful dashboard designed for team oversight:

    Key Sections: - Team Status - Real-time view of who's checked in/out - Today's Summary - Attendance statistics for the current day - Pending Approvals - Leave requests awaiting your decision - Quick Actions - Fast access to common admin tasks - Recent Activity - Latest team check-ins and system events

    Employee Management¶

    Adding New Employees¶

    Bring new team members into the system:

    1. Navigate to Employees section
    2. Click "Add Employee" button
    3. Fill in employee details:
    4. Full name and contact information
    5. Employee ID (auto-generated or custom)
    6. Job title and department
    7. Start date and employment type
    8. Work schedule assignment
    9. Set permissions and role level
    10. Send invitation via email or provide login credentials
    11. Employee receives setup instructions

    Managing Employee Profiles¶

    Keep employee information current and accurate:

    Profile Management: - Edit personal information - Update contact details, addresses - Modify work details - Change job titles, departments, schedules - Adjust permissions - Update role levels and access rights - Upload documents - Store contracts, certifications, photos - Track employment history - Promotions, transfers, role changes

    Bulk Operations: - Import employees from CSV files - Bulk schedule changes for multiple employees - Mass permission updates for role changes - Export employee data for HR systems

    Employee Status Management¶

    Monitor and update employee statuses:

    Status Types: - Active - Currently employed and using the system - Inactive - Temporarily not working (leave of absence) - Terminated - No longer with the company - Suspended - Temporarily restricted access

    Status Actions: - Deactivate accounts when employees leave - Reactivate accounts for returning employees - Suspend access for disciplinary reasons - Transfer between departments or locations

    Real-Time Team Status¶

    Monitor your team's current attendance status:

    Manual Time Adjustments¶

    Correct attendance records when needed:

    When to Use: - Employee forgot to check in/out - System glitches or technical issues - Legitimate work performed outside normal check-in process - Approved overtime or early departures

    Adjustment Process: 1. Find the employee in the attendance records 2. Select the date that needs correction 3. Choose adjustment type: - Add missing check-in - Add missing check-out - Modify existing times - Add break periods 4. Enter correct times and add notes explaining the change 5. Save adjustment - employee is notified of the change

    Attendance Alerts¶

    Stay informed about attendance issues:

    Alert Types: - Late arrivals - Employees checking in after scheduled time - Early departures - Leaving before scheduled end time - Missing check-outs - Forgot to clock out at end of day - Excessive breaks - Break time exceeding policy limits - Location violations - Check-ins from unauthorized locations

    Alert Management: - Set alert thresholds for different violation types - Choose notification methods (email, in-app, SMS) - Assign alert recipients - who gets notified about what - Create escalation rules for repeated violations

    Reporting and Analytics¶

    Generating Reports¶

    Create comprehensive attendance reports for various needs:

    Daily Reports¶

    Perfect for: Daily operations oversight - Team attendance summary for today - Hours worked by each employee - Late arrivals and early departures - Break time analysis - Overtime tracking

    Weekly Reports¶

    Perfect for: Weekly team meetings and planning - Weekly attendance patterns for all employees - Total hours worked per employee - Attendance rate calculations - Trend analysis compared to previous weeks

    Monthly Reports¶

    Perfect for: Payroll processing and HR reviews - Complete monthly attendance data - Overtime summaries for payroll - Leave usage and remaining balances - Attendance compliance metrics

    Custom Reports¶

    Perfect for: Specific business needs - Date range selection - Any period you need - Employee filtering - Specific individuals or departments - Data customization - Choose which metrics to include - Format options - PDF, Excel, or CSV export

    Report Filtering and Customization¶

    Tailor reports to your specific needs:

    Filter Options: - Date ranges - Specific periods or rolling windows - Departments - Focus on specific teams - Locations - Multi-site businesses - Employee types - Full-time, part-time, contractors - Attendance status - Present, absent, late, etc.

    Customization Features: - Choose data columns to include or exclude - Sort options - By name, department, hours, etc. - Grouping options - Group by department, location, or date - Summary statistics - Totals, averages, percentages - Visual elements - Charts and graphs in reports

    Scheduled Reports¶

    Automate report generation and delivery:

    Setup Process: 1. Create report template with desired filters and format 2. Set schedule - Daily, weekly, monthly, or custom intervals 3. Choose recipients - Email addresses for automatic delivery 4. Select format - PDF for viewing, Excel for analysis 5. Activate schedule - Reports generate and send automatically

    Benefits: - Consistent reporting without manual effort - Stakeholder updates delivered on time - Historical tracking with archived reports - Reduced administrative burden

    Leave Request Management¶

    Approval Workflow¶

    Efficiently handle employee time-off requests:

    Reviewing Requests¶

    1. Access Approvals section from admin dashboard
    2. See pending requests with employee details
    3. Review request information:
    4. Employee name and department
    5. Requested dates and duration
    6. Leave type (vacation, sick, personal)
    7. Employee's note or reason
    8. Current leave balance
    9. Check team coverage for requested dates
    10. Make approval decision

    Approval Actions¶

    • Approve - Grant the time off request
    • Deny - Reject with reason for denial
    • Partial approval - Approve some but not all requested days
    • Request more info - Ask employee for additional details
    • Defer decision - Mark for later review

    Approval Notifications¶

    • Automatic emails sent to employees with decision
    • Calendar updates for approved time off
    • Team notifications about approved absences
    • Manager alerts for coverage planning

    Leave Balance Management¶

    Track and manage employee time-off balances:

    Balance Types: - Vacation days - Annual vacation allowance - Sick leave - Health-related time off - Personal days - Personal time allowance - Floating holidays - Flexible holiday time - Comp time - Compensatory time off

    Balance Operations: - View current balances for all employees - Adjust balances for corrections or bonuses - Set accrual rates based on company policy - Handle carryovers from previous years - Track usage patterns and trends

    Leave Policies Configuration¶

    Set up leave policies that match your company rules:

    Policy Settings: - Accrual rates - How much leave employees earn per period - Maximum balances - Caps on accumulated time off - Carryover rules - How unused time transfers to next year - Blackout periods - Times when leave is restricted - Approval requirements - Who can approve different leave types

    Advanced Policies: - Probationary periods - When new employees can start using leave - Seniority-based accrual - Different rates for tenure levels - Department-specific rules - Varying policies by team - Integration with holidays - How company holidays affect leave

    Organization Settings¶

    Company Configuration¶

    Set up your organization's basic information and preferences:

    Company Profile¶

    • Organization name and legal entity information
    • Industry type and business classification
    • Time zone and regional settings
    • Company logo and branding elements
    • Contact information and addresses

    Work Schedule Configuration¶

    • Standard work hours - Default start and end times
    • Work days - Which days of the week are work days
    • Break policies - Required breaks and meal periods
    • Overtime rules - When overtime kicks in and rates
    • Holiday calendar - Company holidays and observances

    Location Management¶

    Configure work locations and geofencing:

    Adding Locations¶

    1. Navigate to Locations section
    2. Click "Add Location" button
    3. Enter location details:
    4. Location name and description
    5. Full address
    6. GPS coordinates (auto-detected)
    7. Allowed check-in radius
    8. Set location-specific rules:
    9. Required for check-in/out
    10. Different work schedules
    11. Special policies
    12. Save and activate the location

    Geofencing Setup¶

    • Set radius for each location (50m to 500m)
    • Test accuracy with sample check-ins
    • Adjust for building size and parking areas
    • Handle GPS drift with appropriate buffer zones
    • Multiple zones for large campuses

    User Roles and Permissions¶

    Control who can access what features:

    Default Roles¶

    • Super Admin - Full system access and configuration
    • Admin - Employee management and reporting
    • Manager - Team oversight and approvals
    • Supervisor - Limited team management
    • Employee - Basic attendance features only

    Custom Roles¶

    Create roles tailored to your organization: - Define permissions for each feature area - Set data access levels - own data, team data, or all data - Configure approval authority - what they can approve - Assign multiple roles to users as needed

    Permission Categories¶

    • Employee Management - Add, edit, deactivate employees
    • Attendance Oversight - View and modify attendance records
    • Reporting Access - Generate and view different report types
    • System Configuration - Change settings and policies
    • Approval Authority - Approve leave requests and adjustments

    Bulk Data Operations: - Export attendance data for payroll processing - Generate multiple reports for different departments - Send notifications to groups of employees - Apply policy changes across the organization

    Troubleshooting and Support¶

    Common Admin Issues¶

    Employee can't check in: - Verify employee is active and has proper permissions - Check location settings and GPS accuracy - Confirm employee is using correct login credentials - Review any active restrictions or suspensions

    Reports showing incorrect data: - Check date ranges and filters applied - Verify employee assignments to correct departments/locations - Review any manual time adjustments that might affect totals - Ensure all employees have completed their check-out process

    Leave requests not appearing: - Confirm employees are submitting to correct approval chain - Check spam folders for approval notification emails - Verify leave policy settings allow requests for those dates - Review employee leave balances and eligibility
