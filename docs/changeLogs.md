# Release Notes

Stay up to date with the latest features, improvements, and fixes in My Attend. We're constantly working to make your attendance tracking experience better.

## Version 1.0.6 (iOS) | Version 1.5.7 (Android)
**Released: December 2025**

### Fixed Bugs
- **Fixed issue in Activity module** that caused incorrect behavior in some cases
- **Fix attachment at request module** - improved reliability and functionality

### What's New?

#### Profile Page
- **Added a new Profile page** for users
- **Displays employee information** and basic account details
- **Prepared for future profile management features**

#### Attendance Log
- **Added validation to control** whether photo is required during attendance
- **Photo requirement is configurable** based on company settings
- **Photo requirement can be set** per employee or team
- **Hide photo if it does not exist** or is optional
- **Added new animated GIF** to enhance user feedback on attendance actions

#### Activity View
- **Fixed activity-related bug**
- **Improved activity flow** and data display
- **Ensures correct behavior** between attendance and requests

#### Splash Screen
- **Added a new splash screen** on app launch
- **Displays branding** and improves initial loading experience

#### Benefit and Deductions
- **Enhanced calculation logic** to ensure accurate totals
- **Added effective date and notes fields** for better tracking
- **Improved UI** for easier editing and review by users
- **Ensures consistency** with company payroll rules

### Impact
- **Enhances overall UX** and visual feedback
- **Adds core screens**: Profile and Splash
- **Improves attendance logic** with flexible photo validation
- **Improves attachment usability** and reliability
- **Increases system stability** with bug fixes

## System Requirements

### iOS Requirements
- **Operating System**: iOS 15.0 or later
- **Compatible Devices**: iPhone 6s and newer, iPad Air 2 and newer
- **Storage Space**: 50MB available space
- **Internet**: Required for initial setup and data synchronization

### Android Requirements
- **Operating System**: Android 6.0 (API level 23) or later
- **RAM**: 2GB minimum, 4GB recommended
- **Storage Space**: 50MB available space
- **Internet**: Required for initial setup and data synchronization
